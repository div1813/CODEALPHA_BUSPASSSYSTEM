# 🚌 Cloud-Based Bus Pass System | CodeAlpha Internship Task 3

A cloud-hosted online bus ticket booking system built with HTML, CSS, and JavaScript.
Deployed on GitHub Pages — no backend required!

---

## 🌐 Live Demo
[Click here to view the live project](https://div1813.github.io/CODEALPHA_BUSPASSSYSTEM)

---

## 📌 Features

- 🗺️ **Multiple Bus Routes** — 5 routes with duration and pricing
- 💺 **Visual Seat Selection** — interactive seat map with real-time availability
- 👤 **Passenger Details** — name, age, phone number and travel date
- 🎫 **Digital Ticket** — auto-generated unique ticket ID for every booking
- 📋 **My Tickets** — view all previously booked tickets
- ☁️ **Cloud Hosted** — deployed on GitHub Pages (accessible from anywhere)
- 📱 **Responsive Design** — works on mobile and desktop

---

## 🛡️ Cloud Computing Concepts Used

| Concept | Implementation |
|---------|---------------|
| **Cloud Hosting** | Deployed on GitHub Pages |
| **Scalability** | Static app scales automatically with GitHub CDN |
| **High Availability** | GitHub Pages ensures 99.9% uptime |
| **Data Storage** | LocalStorage simulates cloud database |
| **Unique ID Generation** | Timestamp-based ticket ID system |
